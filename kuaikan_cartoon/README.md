# React全家桶仿PC端《快看漫画》（部分功能）

## 简介 
- 简单React项目，实现“登录/退出”、“关注/取关”、“关键词搜索”等功能。
<br><br>
## 启动
进入kuaikan_cartoon目录

    npm cache clean --force
    npm install
    yarn start
